# v0.3.0 한국어 공개 안내문

React Native Image Compression Kit v0.3.0을 공개했습니다.

Android와 iOS에서 JPEG, PNG, 런타임 지원 여부를 확인한 WebP 압축을
제공하며, 리사이즈·목표 용량·메타데이터 정책·타입이 있는 오류·capability
API를 함께 제공합니다. 이번 릴리스는 네이티브 압축 동작을 바꾸기보다
도입 과정을 검증 가능하게 만드는 데 집중했습니다. 공개 연동 문서, 네이티브
예제, Bare React Native와 Expo development build 호환성 검증, 커뮤니티
지원 경로, provenance가 있는 npm 배포를 포함합니다.

```sh
npm install react-native-image-compression-kit@0.3.0
```

지원 범위는 Android 23 이상, iOS 13.4 이상입니다. Expo는 development
build가 필요하며 Expo Go는 지원하지 않습니다. HEIC, HEIF, AVIF 출력은
아직 구현하지 않았습니다.
