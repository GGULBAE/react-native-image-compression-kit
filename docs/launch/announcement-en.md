# v0.3.0 English announcement

React Native Image Compression Kit v0.3.0 is available.

It provides native JPEG, PNG, and capability-gated WebP compression for Android
and iOS, with resize, target-size, metadata, typed errors, and runtime capability
APIs. This release focuses on making adoption verifiable: a public integration
site, a native example, Bare React Native and Expo development-build compatibility
lanes, community support paths, and provenance-backed npm publishing.

```sh
npm install react-native-image-compression-kit@0.3.0
```

Android 23+ and iOS 13.4+ are supported. Expo requires a development build;
Expo Go is not supported. HEIC, HEIF, and AVIF output are not implemented.

- Documentation: https://ggulbae.github.io/react-native-image-compression-kit/
- Source and support: https://github.com/GGULBAE/react-native-image-compression-kit
- Release: https://github.com/GGULBAE/react-native-image-compression-kit/releases/tag/v0.3.0

Feedback is welcome through Discussions, and reproducible defects through the
issue templates. Please use private vulnerability reporting for security issues.
